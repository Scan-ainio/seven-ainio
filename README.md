# Takken Scan Coach

日本宅建士小吴 Coach 学习项目。

这个项目的目标不是做一个复杂软件，而是在未来 12 周内帮助我更高效率地通过宅建士考试。项目会优先使用 Markdown 管理知识、错题、每日记录、模拟考试和考前策略，之后再按需要逐步增加网页、自动化或小吴 Coach 功能。

## 核心原则

- 以考试通过为第一目标。
- 先建立学习系统，再开发工具。
- 内容优先，功能从小处开始。
- 每天可执行，每周可复盘。
- 高频考点优先，错题反复强化。
- 文件结构清晰，长期可维护。

## 当前阶段

第一阶段只做项目规划、目录设计和基础文件建立。

暂不开发网页，不连接外部 API，不写复杂程序。

## 核心模块

- 知识库：整理宅建士考试知识点。
- 每日学习计划：安排每天要学什么、练什么、复盘什么。
- 错题本：记录错因、正确思路和复习日期。
- 题库与练习：保存主题练习题和历年题整理。
- 模拟考试：管理整套模拟考试、得分和弱点分析。
- 高频考点：按考试频率和得分价值排序。
- 术语表：解释法律、交易和不动产相关关键词。
- 记忆强化：用口诀、对比表、案例和间隔复习帮助记忆。
- 考前策略：管理时间分配、做题顺序和最后冲刺重点。
