# Student Profile 模板

## 使用方法

这个文件是小吴 Coach 学习教练的基础档案。建议复制一份命名为 `student-profile.md`，每周更新一次；如果做了模拟考试，也要当天更新。

填写原则：

- 不追求完美，先填当前真实状态。
- 掌握度用百分比或 1-5 分都可以，但建议保持同一种方式。
- 小吴当前判断要写得直接：现在最危险的是什么，最该拿分的是什么。
